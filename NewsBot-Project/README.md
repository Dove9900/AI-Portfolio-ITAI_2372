NewsBot Project

This folder contains a lightweight news-analysis application built in Python.
The goal of the project was to explore basic Natural Language Processing (NLP) concepts using custom utility functions and simple text-classification techniques.

What the NewsBot Does

Preprocesses and cleans text

Extracts TF-IDF features

Classifies news content

Analyzes sentiment

Identifies named entities

Displays POS tags and dependency relations

Presents results through a Streamlit interface

Skills Demonstrated

Python scripting

Basic NLP techniques

Organizing multi-file Python projects

Using helper modules (utils)

Understanding how data preprocessing affects model output
